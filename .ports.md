# Port Configuration

## Development Ports

- **Web Dashboard**: http://localhost:3000
- **API Gateway**: http://localhost:5100
- **Market Data Service**: http://localhost:5001 (gRPC)

## Starting the Services

Run each in a separate terminal:

```bash
# Terminal 1 - Market Data Service (gRPC)
yarn dev:market-data

# Terminal 2 - API Gateway (REST/SignalR)
yarn dev:api-gateway

# Terminal 3 - Web Dashboard (React)
yarn dev:web
```

Or just the frontend:
```bash
yarn dev
```
